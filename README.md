# IBA App Dev Project

## Description
This is the Android application for the IBA App Dev Project.  
The app helps with the seamless connection between rider providers and rider takers without sharing their phone number. 

## Features
- View and manage data (create, read, update, delete)  
- Simple and intuitive user interface  

## Tech Stack
- Language: Dart (RiverPod)
- Platform: Android  
- Build system: Android Studio  

## Installation
1. Clone the repository:  
    ```bash
    git clone https://github.com/Vishal-Das-01/IBA_App_Dev_Project.git
    cd IBA_App_Dev_Project/project_app_dev
    ```
